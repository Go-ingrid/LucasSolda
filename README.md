O sistema desenvolvido consiste em um site institucional para divulgação de serviços de soldagem e fabricação de estruturas metálicas.
A plataforma foi projetada para apresentar de forma clara os serviços oferecidos pelo profissional, incluindo estruturas metálicas, portões e grades, manutenção e reparos 
e projetos personalizados.
O site possui uma estrutura simples e objetiva, contendo seções como apresentação do serviço, descrição das soluções oferecidas, área de contato e integração com WhatsApp 
para facilitar a solicitação de orçamentos.
O objetivo principal do sistema é funcionar como uma vitrine digital para o profissional, permitindo que clientes conheçam os serviços e entrem em contato de forma rápida.

  • Tecnologias utilizadas
-HTML5
-CSS3
-JavaScript
-GitHub

  • Funcionalidades implementadas
-Estrutura de site institucional
-Apresentação dos serviços oferecidos
-Layout organizado com seções informativas
-Botão de contato via WhatsApp
-Formulário de contato para solicitação de orçamento
-Exibição de imagens e vídeos dos serviços realizados
-Navegação entre seções da página

  • Procedimentos de Execução Local
-Clonar o repositório do projeto no GitHub ou baixar o arquivo .zip.
-Acessar a pasta principal do projeto.
-Abrir o arquivo index.html em qualquer navegador web.

O sistema não necessita de servidor ou dependências externas para funcionamento básico.
